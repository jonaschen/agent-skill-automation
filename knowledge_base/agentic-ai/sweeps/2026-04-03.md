## Cross-Cutting Analysis

- **Agent payments infrastructure emerging**: x402 (MCP-native payments) is the first credible standard for AI agents to transact autonomously. Backed by Visa, Mastercard, Coinbase, Stripe, AWS. This addresses the missing economic layer — agents can now discover tools (MCP), coordinate (A2A), and pay (x402).
- **MCP vs A2A convergence accelerating**: MCP Tasks primitive (SEP-1686) moves toward agent-to-agent patterns while A2A's Agent Engine integration provides managed deployment. Both are in production hardening. x402 adds a payment layer on top.
- **Enterprise agent identity**: A2A's Signed Agent Cards and MCP's DPoP/WIF security SEPs both address the "who is this agent?" problem — confirming agent identity is the #1 enterprise blocker. Different approaches: A2A uses cryptographic card signing, MCP uses standard OAuth/OIDC primitives.
- **Anthropic security posture under scrutiny**: Two major leaks in one week (Mythos data leak Mar 26, source code leak Mar 31) plus the deny-rule bypass disclosure. The source code leak revealed anti-distillation defenses and autonomous daemon mode (KAIROS). Contrast with Google's security narrative (Signed Agent Cards, CAA, CMEK, sandboxed VMs).
- **On-device agent explosion**: Gemma 4 E2B (<1.5GB) + Gemini Nano 4 on 140M devices + ADK multi-language parity + Astra glasses = Google building a complete edge-to-cloud agent stack. Anthropic has no on-device story.
- **Agentic Vision paradigm**: Gemini 3 Flash's Think-Act-Observe loop for images parallels the agentic coding loop (plan-code-test). This pattern of "make X agentic via iterative code execution" is likely to expand to more modalities.
