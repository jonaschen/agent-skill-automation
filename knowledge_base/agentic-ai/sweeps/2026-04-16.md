## Anthropic Updates

### Claude Code (v2.1.105 → v2.1.109)

Four releases in 3 days (April 13-15):

- **v2.1.109** (Apr 15): Rotating progress hint for extended-thinking indicator — improved user visibility during long reasoning passes
- **v2.1.108** (Apr 14, major):
  - **Prompt caching TTL controls**: `ENABLE_PROMPT_CACHING_1H` env var opts into 1-hour cache TTL; `FORCE_PROMPT_CACHING_5M` forces 5-minute TTL; startup warning if caching disabled
  - **Session recap**: `/recap` command provides context summary on return to session; auto-triggered after away periods; `CLAUDE_CODE_ENABLE_AWAY_SUMMARY` force-enables
  - **Skill tool discovers built-in slash commands**: `/init`, `/review`, `/security-review` now invocable via Skill tool alongside user skills — unifies skill discovery
  - `/undo` aliases `/rewind`; enhanced `/model` warns before mid-conversation model switch; improved `/resume` picker; errors distinguish rate limit vs plan usage limit
  - Language grammars load on-demand → lower memory footprint; paste regression in `/login` fixed; Agent tool prompting fix in auto mode
- **v2.1.107** (Apr 14, minor): Thinking hints shown sooner during long operations
- **v2.1.105** (Apr 13):
  - **PreCompact hook**: hooks can block compaction (exit code 2 or `{"decision":"block"}`)
  - **Background Monitor support**: plugins declare `monitors` manifest key for auto-arming at session start or skill invoke
  - **EnterWorktree path parameter**: switch into existing worktree by path (not just create new)
  - **Skill description cap raised 250→1,536 characters** with startup warning on truncation
  - **WebFetch strips `<style>` and `<script>` tags** — prevents CSS/JS bloat in fetched content
  - **Stalled API stream abort**: 5-minute no-data timeout with non-streaming retry
  - `/proactive` aliases `/loop`; MCP output truncation with format-specific recipes (jq for JSON)
  - **Bug fixes**: images dropped from queued messages; leading whitespace trimmed from assistant messages (breaking ASCII art); garbled bash output with rich/loguru links; marketplace plugins with `package.json` not auto-installing; 429 showing raw JSON; washed-out colors over SSH

Sources: https://code.claude.com/docs/en/changelog, https://releasebot.io/updates/anthropic/claude-code

### Agent SDK

- **Python v0.1.59** (Apr 13): Maintenance release bundling CLI v2.1.105. No new Python APIs. Still 4 versions behind CLI v2.1.109 — expect v0.1.60 when next bundle prepared.
- **TypeScript SDK V2 interface (unstable preview)**: Significantly simplified API — removes async generators entirely; each turn is separate `send()`/`stream()` cycle. Three-concept API: `unstable_v2_createSession()` / `unstable_v2_resumeSession()` / `unstable_v2_prompt()`. `SDKSession.send(message)`, `stream() → AsyncGenerator<SDKMessage>`, `close()`. Session lifecycle uses TypeScript 5.2+ `await using` for auto-cleanup. Session IDs extractable from any `SDKMessage.session_id` for cross-session resume. All V2 exports prefixed `unstable_v2_*`. No GA timeline. Available in existing `@anthropic-ai/claude-agent-sdk` package.

Sources: https://github.com/anthropics/claude-agent-sdk-python/releases (v0.1.59), https://code.claude.com/docs/en/agent-sdk/typescript-v2-preview

## Cross-Cutting Analysis

- **CLI context management divergence deepens**: Gemini CLI v0.38.0 ships context compression as an explicit, user-visible service; Claude Code's compaction is system-managed and now hook-gatable (PreCompact hook). Google's approach gives user visibility into compression events; Anthropic's gives pipeline control. The v0.39.0 `/memory inbox` passive skill lifecycle (background extraction → user approval → activation) is more sophisticated than Claude Code's explicit `/memory` commands — Google is building toward frictionless background skill capture.

- **TTS as agentic capability gap**: Gemini 3.1 Flash TTS (Elo 1211, audio tags, multi-speaker, 70+ languages) represents a capability class Anthropic has no equivalent for. Voice agents, call center automation, audio content generation — all first-class on Gemini, third-party-required on Claude. This is the clearest Anthropic-Google asymmetry in the current period.

- **Claude Mythos on Vertex AI**: Cross-vendor distribution of Claude Mythos via Google Cloud is strategically notable — Anthropic's most powerful model is available on a Google infrastructure product before broader public access. Validates the Anthropic-Google Cloud partnership depth. At $25/$125/MTok it's 5x Opus 4.6 pricing — this creates a new premium tier.

- **Open stack completion**: ADK Java 1.1.0 + ADK Python v1.30.0 (Gemma 4) = Google offers open-weight (Gemma 4) + open-framework (ADK multi-language) + managed-runtime (Agent Engine) — a fully open-to-managed agent stack. Anthropic's equivalent requires proprietary Claude API + open Agent SDK + third-party hosting.

- **Skill description cap increase** (250→1,536 chars): Directly impacts our pipeline. Richer trigger descriptions improve routing accuracy; we should revisit all SKILL.md description fields to expand and sharpen.

- **Google I/O 33 days**: Pre-announcement window opens ~May 2 (17 days). Firebase Studio + ADK focus confirmed; expected announcements: ADK v2.0, AI Applications relaunch, Gemini 4, Android XR/Samsung glasses formal announcement, Mariner Studio preview.

## Implications for Our Pipeline

- **P0 — Skill description cap**: Raise all SKILL.md trigger descriptions from the old 250-char limit toward the new 1,536-char cap. Richer descriptions improve routing accuracy directly. Current crop likely under-specified.
- **P0 — Haiku 3 retirement in 3 days**: Verify deprecation guard fires; no agent configs should reference `claude-3-haiku-20240307`.
- **P1 — Prompt caching TTL**: `ENABLE_PROMPT_CACHING_1H` is directly applicable to our steward agents. Nightly runs exceeding 5 minutes will benefit significantly from 1-hour cache TTL. Add to all steward launch scripts.
- **P1 — PreCompact hook**: Evaluate adding PreCompact hook to steward agents to prevent premature compaction during critical research/writing phases. Control compaction timing explicitly.
- **P1 — Background Monitor**: The `monitors` manifest key enables plugins to auto-arm at session start — useful for Phase 5 supervisor/watchdog agents.
- **P1 — Claude Mythos on Vertex AI at $25/$125/MTok**: Update Phase 7 AaaS cost model. Mythos is 5x Opus 4.6 — must include this tier in billing design.
- **P2 — TypeScript SDK V2 preview**: Evaluate for Phase 5 TypeScript-based orchestrators. The `send()`/`stream()` model is cleaner for injecting pipeline logic between turns. Do not use in production yet (`unstable_v2_*`).
- **P2 — ADK Java 1.1.0 + Firebase Studio**: Monitor Firebase Studio for cloud-based agentic IDE capabilities relevant to Phase 5/7 architecture. ADK Java HITL (ToolConfirmation) pattern is a reference for our own HITL design.
- **P2 — Gemini 3.1 Flash TTS**: Document as P2 gap — no TTS capability in current agent fleet. Not blocking for Phases 1-5, but relevant for Phase 7 AaaS voice agent use case.
- **P3 — Google I/O monitoring**: Begin tracking pre-I/O announcements in 17 days. Key watch items: ADK v2.0, A2A v1.1, Gemini 4, AI Applications platform refresh.
