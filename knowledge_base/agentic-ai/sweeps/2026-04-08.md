## Cross-Cutting Analysis
- **Both vendors in extended stabilization**: Anthropic focused on Mythos/Glasswing rollout and deprecation management; Google focused on Google I/O preparations. Neither has shipped new code in 4+ days — longest pause since the March release surge began.
- **Security-first model deployment is the new paradigm**: Anthropic's Glasswing (40+ security partners) + Google's CodeMender (72 upstream fixes) + AWS MCP IAM context keys all point to security as the primary gating function for frontier model access. This is a structural shift from "ship fast, secure later."
- **CLI agents converging architecturally**: Claude Code and Gemini CLI now share virtually identical architecture patterns (MCP, project config files, subagent definitions, skill loading, worktree isolation). The differentiation is in model quality, not tooling.
- **Open model competitive landscape heating up**: Qwen 3.6 Plus disrupts the Gemini vs Claude agentic coding narrative by offering competitive SWE-bench scores at 3x speed and free access. Gemma 4's speed disadvantage (3.2x slower than Qwen 3.5) limits its server-side agentic appeal despite quality leads.
- **Multiple deprecation deadlines converging**: Haiku 3 (Apr 19), 1M context beta (Apr 30), Sonnet 3.7 (May 11), Haiku 3.5 (Jul 5) — organizations need migration plans.
