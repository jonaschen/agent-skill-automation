## Cross-Cutting Analysis
- **ADK v1.31.0 Session Rewind parallels Anthropic's implicit "undo" in Claude Code** (git worktree isolation, branch-based rollback). Both ecosystems are converging on the principle that agent actions must be reversible. Google's approach is session-level; Anthropic's is git-level.
- **Computer Use parity**: Google (gemini-3-pro/flash-preview `computer_use` tool) now matches Anthropic (Opus 4.7 computer use). Both are in preview/beta. Google has the additional Mariner consumer product layer.
- **Agent Engine features routing through ADK**: Google is using ADK releases to ship Agent Engine capabilities during the 59-day Agent Builder freeze. This is a notable deployment pattern — new cloud features arriving via client SDK updates rather than platform release notes.
- **Pre-I/O freeze deepens across all Google products**: CLI (3d quiet), API (3d quiet), ADK (v1.31.0 shipped but v2.0 held), A2A (37d), Agent Builder (59d), Mariner (13d), Astra (13d), Gemma (16d). The freeze is now the deepest and broadest observed.
