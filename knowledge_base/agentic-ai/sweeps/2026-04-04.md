## Cross-Cutting Analysis

- **MCP + A2A convergence accelerating**: MCP Dev Summit had OpenAI presenting, both protocols now under Linux Foundation governance. MCP Resources alignment between OpenAI and Anthropic SDKs creates de facto interoperability standard.
- **Desktop agent arms race**: Claude Computer Use now on Mac + Windows (consumer desktop); Google Mariner in cloud VMs; Gemini CLI challenging Claude Code at the terminal. All three major players now have consumer-facing desktop automation.
