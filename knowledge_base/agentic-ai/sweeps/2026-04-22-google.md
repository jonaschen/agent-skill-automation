## Cross-Cutting Analysis

- **Deep Research Agent is Google's answer to Anthropic's Managed Agents research mode.** Both offer autonomous agent execution behind an API. Key difference: Google's agents have native MCP server integration (connecting to external tools), while Anthropic's Managed Agents provide built-in tools and MCP proxies. Google's pricing ($1-7/task) vs Anthropic's Managed Agents ($0.08/session-hr + tokens) suggests different billing models for agentic workloads.
- **The Interactions API is becoming Google's agentic API surface.** Deep Research, ADK Agent Engine, and potentially future agent types all route through this API rather than the standard `generateContent` endpoint. This is analogous to Anthropic's distinction between Messages API and Managed Agents API.
- **CLI freeze is the deepest observed** (day 5 stable, day 7+ nightly). Combined with API silence broken only by Deep Research, suggests Google teams are in focused pre-I/O preparation mode.
