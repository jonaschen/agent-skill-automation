## Cross-Cutting Analysis
- **Stabilization convergence**: Both Anthropic and Google entered simultaneous stabilization phases after intense March/early April release bursts. Neither vendor has released new versions in 3+ days — unusual given the ~daily cadence of recent weeks. This suggests both are preparing for their next major push (Anthropic: possible v2.2.x series; Google: I/O 2026 announcements).
- **MCP security maturation**: The OWASP MCP Top 10, Adversa AI TOP 25, and CoSAI white paper now provide three independent security frameworks for MCP. The 658x cost amplification attack is a new threat category beyond poisoning — financial attacks rather than data exfiltration.
- **Agent CLI convergence**: Gemini CLI and Claude Code continue converging on nearly identical architectures (MCP, project config files, markdown agent definitions, tool isolation, worktree-based parallelism). Gemini CLI's passive skill extraction via background memory is a differentiator Claude Code lacks.
- **Implicit vs explicit memory**: Gemini CLI's background memory service (passive skill extraction from usage patterns) represents a philosophically different approach from Claude Code's explicit memory ("remember X"). Both approaches have merit; implicit risks capturing noise while explicit risks missing patterns.
- **Geopolitical risk for Anthropic**: The US blacklisting saga introduces regulatory uncertainty for government customers. If the Ninth Circuit appeal succeeds, federal agencies may be forced to use Google or OpenAI instead. This doesn't affect commercial users but could shape enterprise procurement decisions.
