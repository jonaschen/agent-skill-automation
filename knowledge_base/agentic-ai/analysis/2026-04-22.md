## 1. Gap Analysis

### 1.1. Session Rollback (The "Rewind" Gap)
- **Finding**: ADK v1.31.1 and Agent SDK v1.31.0 both shipped **Session Rewind**.
- **Impact**: Our Phase 4 optimizer currently uses a "blind retry" model which is token-inefficient and prone to oscillating between failures.
- **Priority**: **P1**. We must adopt the Rewind pattern to enable "branching search" in our description optimization loop.

### 1.2. Memory Persistence (The "Episodic" Gap)
- **Finding**: Vertex AI launched **Memory Bank GA** for episodic memory; Gemini CLI v0.39 added a **Background Memory Service**.
- **Impact**: Our orchestrators are stateless. They don't remember "lessons learned" from previous days' factory runs.
- **Priority**: **P1**. Integrating a managed episodic memory (Memory Bank) into Phase 5 will prevent the factory from repeating the same architectural mistakes across sessions.

### 1.3. Evaluation Crisis (The "Opus 4.7 Regression")
- **Finding**: Shadow Eval reveals a **trigger regression (0.63 vs 0.829)** for Opus 4.7.
- **Impact**: 4.7 is "too literal" for our current Skill trigger descriptions.
- **Priority**: **P0**. We cannot cut over to 4.7 until the `autoresearch-optimizer` completes a tuning pass on all core Skills to match 4.7's literal style.

## 3. Threat Detection

### 3.1. Role Escalation (ChatInject Vulnerability)
- **Finding**: **ChatInject (arXiv:2509.22830)** demonstrates role-token injection in tool outputs.
- **Threat**: A compromised tool (or malicious web page fetched via WebFetch) could inject `<|system|>` tokens into the context, escalating privileges.
- **Action**: **P0 Audit**. All tool outputs in our pipeline must be sanitized at the token level before being returned to the model.

### 3.2. Architecture Obsolescence (Managed Agents)
- **Finding**: Anthropic's **Managed Agents** platform provides a fully managed harness that competes with our Phase 5/7 goals.
- **Threat**: Our custom orchestration layer may become redundant if Managed Agents adds better multi-agent support.
- **Mitigation**: Focus on **Model-Agnostic Orchestration** (S3). Our value is in the "Changeling" cross-platform parity, which Managed Agents (Anthropic-only) cannot offer.
